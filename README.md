# Kats
KATS (Kiwoom Automated Trading System)

Kiwoom is a securities company in Korea.
Kats is a program that implements automatic trading using Kiwoom Securities' OpenAPI.
If you are interested in the development, please contribute.

Visual Studio 2017, .Net Framework 4.7.1 Winform is being developed.
Supports automatic patch file creation
Call ".\_Patch\_Kats_NoCI_Build_Patch.cmd" to create patch file.

There is a separate automatic version-up system
This has not been released to Git yet.
We need document work,
If you are interested, we will reply if you contact us.

키움은 한국의 증권 회사입니다.
Kats는 키움증권의 OpenAPI를 이용한 자동 매매를 구현한 프로그램입니다.

.\_Patch\_Kats_NoCI_Build_Patch.cmd 로 호출하면 생성됩니다.

별도의 자동 버전 증가 시스템이 있으며
이는 아직 Git에 공개하지 않았습니다.
문서 작업등이 필요하며
관심 있으신분은 문의 주시면 답해드리겠습니다.
